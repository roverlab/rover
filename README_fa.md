**Rover** یک کلاینت دسکتاپ مدرن برای sing-box است که شما را از نوشتن دستی و خسته‌کننده پیکربندی‌های JSON راحت کرده و تجربه پراکسی **WYSIWYG** (آنچه می‌بینید همان چیزی است که دریافت می‌کنید) واقعی را ارائه می‌دهد.

## ✨ ویژگی‌های کلیدی

- 🎨 **کاملاً بصری**: ویرایش گرافیکی **قوانین مسیریابی** و **DNS سفارشی**، با پشتیبانی از پروتکل‌های متعدد و مسدودسازی تبلیغات.
- 🛡️ **حالت TUN**: پشتیبانی بومی از رابط‌های شبکه مجازی TUN، امکان تسلط یک‌کلیکی بر ترافیک کل سیستم.
- 🔄 **سازگاری کامل با Clash**: سازگاری کامل با اشتراک‌ها و مجموعه قوانین Clash / Mihomo، مهاجرت بدون هزینه برای کاربران فعلی.
- 📦 **آماده استفاده**: قالب‌های مسیریابی کلاسیک داخلی (مانند لیست سفید داخلی / حالت جهانی)، حذف پیکربندی پیچیده.
- 📊 **مانیتورینگ بلادرنگ**: مشاهده سرعت آپلود/دانلود، تأخیر گره‌ها و اتصالات فعال به صورت بلادرنگ.

## 🌐 پروتکل‌های پشتیبانی‌شده
`Shadowsocks` / `VMess` / `VLESS (Reality)` / `Trojan` / `Hysteria2` / `TUIC` / `AnyTLS` / `HTTP/SOCKS5`

## ⚡ شروع سریع

1. **دانلود**: به صفحه [Releases](https://github.com/roverlab/rover/releases) مراجعه کنید تا نسخه‌های Windows / macOS / Linux را دریافت کنید.
2. **وارد کردن**: لینک اشتراک خود را در صفحه «پروفایل‌ها» جای‌گذاری کنید یا یک فایل محلی را وارد کنید.
3. **اتصال**: یک قالب داخلی را در صفحه «سیاست‌ها» اعمال کنید، سپس پراکسی (یا حالت TUN) را فعال کنید تا شروع کنید.

## 🛠️ توسعه و ساخت

</div>

```
npm install
npm run dev        # شروع توسعه
npm run pack:win   # ساخت نسخه Windows
```
